## 한 줄

AI.SW Academy 학회 활동에서 Word Vector, Language Model, CNN, RNN, Attention, Transformer를 공부하며, **GitHub Pages 기반 웹북**을 공동 산출물로 만들어가고 있습니다. 최종 목표는 한국어 전용 LLM 개발 프로젝트 참여입니다.

## 무엇을 하고 있나

- 학회 커리큘럼: Word Vector → Language Model → CNN → RNN → Attention → Transformer
- 각 주차 학습 내용을 정리해 웹북에 챕터로 기여

## OH! GYM과의 연결

휴머노이드 RL 자체와는 거리가 있어 보이지만, 휴머노이드의 다음 단계는 **language-conditioned 정책 / VLA 모델**입니다. Transformer · Attention을 처음부터 이해하고 있으면, 이 흐름이 들어왔을 때 빠르게 따라갈 수 있다고 생각합니다.

또한 **공동 문서 산출물을 끝까지 만들어본 경험**은 OH! GYM의 "홍보·연구 생태계 구축" 측면과도 자연스럽게 연결됩니다.

## One line

At the AI.SW Academy student association, I'm studying Word Vectors, Language Models, CNN, RNN, Attention, and Transformer, and co-authoring a **GitHub Pages web book** as our joint output. Long-term goal: contribute to a Korean-only LLM project.

## What I do

- Curriculum: Word Vector → Language Model → CNN → RNN → Attention → Transformer
- Contribute weekly chapters to the web book
- Help with GitHub Pages structure, commit workflow, and reviews
- Use it as preparation for the Korean LLM project

## Connection to OH! GYM

It looks distant from humanoid RL, but the next layer of humanoid intelligence is **language-conditioned policies / VLA models**. Understanding Transformers and Attention from the ground up means catching this wave faster when it arrives.

Co-authoring a shared documentation artifact end-to-end also lines up naturally with the "research-ecosystem and outreach" side of the OH! GYM internship.
